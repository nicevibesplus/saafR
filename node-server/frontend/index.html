Safe Accident and Anxiety Free Routing

We developed this application to enable users to navigate safely through Münster by bicycle. Our app offers three main functions:

Routing

Use the routing feature to find the safest bike routes in Münster. In the routing menu, you can choose whether you want to avoid accidents, anxiety zones, or both.

Adding Anxiety Zones

On the "Adding" page, you can mark and edit areas where you feel unsafe while cycling.

About

Learn more about saafR, our team, and how the application works.
